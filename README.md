=============

### Navne Konventioner

Generelt:
- kun på engelsk
- ingen mellemrum
- underscores mellem ord (like_this), men bindestreger i css

#### CSS

```css
#page-header
.class-name
```
Som kan bruges i HTML:
```HTML
<header id="page-header"></header>
```

#### functions

Små bogstaver med underscores imellem.

```php
function name_of_function() 
{
  // function body
}
```
